# 🥛 Dairy Farm Website

A modern and responsive React-based website developed for **Shree Shyam Dairy**. The website is designed to provide customers with information about the dairy farm, its products, facilities, and services through a clean and user-friendly interface.

## Features

* Responsive and modern user interface
* Home, About, Farm, Products, Gallery, and Contact pages
* Product showcase section
* Media gallery for farm-related content
* Smooth navigation using React Router
* Mobile-friendly design

## Tech Stack

* React
* Vite
* React Router DOM
* JavaScript (ES6+)
* CSS

## Getting Started

```bash
npm install
npm run dev
```

Currently under active development, with additional features and content being added.
